# Tubame — Business Landing Page

一個客戶設計的 **店家形象官網 (Landing Page)**，  
重視畫面呈現與響應式體驗。
(僅有畫面，沒有實際功能)

---

## 功能特色
- **現代化設計**：使用 React + BEM CSS，結構清晰易於維護
- **全 RWD**：適配桌機、平板、手機
- **互動動畫**：整合多個 React 動畫套件
- 部署於 Vercel，提供即時 Demo

---

## 技術棧
- **Frontend**：React (Vite) + BEM CSS
- **Animation**：Framer-motion
- **Deploy**：Vercel
